## Installation

1. In your terminal, clone the repo
   ```sh
   git clone https://github.com/GreekGodBod/fe_greek_god_bod
   ```
2. `cd` into that directory
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the server
   ```sh
   npm start
   ```
